# cypresstests
Configuration
Please create cypress.env.json file and fill it with the content

{
   
    "loginCda": "your-cda-login",
    
    "passwordCda": "your-cda-password",
    
    "baseUrl": "https://cda.pl",
   
    "userName": "your-user-name"
}


Install the project
npm install


Run the project
npm run cypress:open
